Программа SPC_maker используется для подготовки начальных данных для электромагнитных расчетов. На вход она получает результаты расчетов программы
'PERENOS'. Ключевым понятием в расчетах, проводимых программой 'PERENOS' и СПК является понятие детектора и связанные с ним понятия семейства детекторов
и спектра детектора. Детектор - площадка, через которую мы фиксируем поток частиц. Рассматриваются два вида площадок: круговые площадки (тогда детектор 
назвается детектором типа детектор) и прямоугольные
площадки, совпадающие с гранями ячеек пространственной сетки(детектор типа грани). Семейство детекторов - набор площадок, детектирующих частицы одного
 вида. Анализируя
направление и энергию частиц после прохождения площадки мы можем рассматривать источник частиц такой, что потоки совпадают. Под спектром этого источника
будем понимать описание потока частиц в таком виде: энергия направление доля. Спектр детектора это спектр соответствующего источника частиц.
Семейства детекторов перечислены и описаны в файле 'detectors'. Описание состоит из 6 строк имя семейства (используемое для именования файлов потока),
тип детектируемой частицы, тип детектора, имя файла с координатами детекторов, имя файла с шаблоном вывода.
'Perenos' работает как с использованием разностной схемы, так и без нее. Нужно отметить некоторые особенности работы программы в каждом из двух случаев: 
при использовании разностной схемы в папке проекта должны находиться файлы, описывающие конфигурацию пространства (файл с описанием пространственной 
сетки и файл с описанием, какая ячейка какому слою принадлежит), если же разностная схема не используется, то эти файлы могут отсутствовать.
Также важно отличие в типах расчитываемых детекторов: детекторы типа грани рассчитываются только с использованием пространственной сетки. 
 В каждом из этих случаев программа 'PERENOS' создает для каждого семейства файл с полным потоком 
(имя_семейства.tot) и файл с подробным потоком (имя_семейства.res). Подробность информации во втором файле зависит от шаблона. 

После того, как программа 'PERENOS' отработала, запускается программа SPC_maker.
